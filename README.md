# Web-Design-HW1
#  Personal CV Website

##  Descriere

Acest proiect reprezintă un **website personal de tip CV/portofoliu**, realizat folosind **HTML și CSS**, având ca scop prezentarea informațiilor personale, a abilităților și a proiectelor realizate.

Website-ul este structurat în mai multe pagini interconectate și oferă o experiență simplă și clară de navigare.

---

##  Obiective

- Crearea unui CV online
- Prezentarea proiectelor personale
- Implementarea unei interfețe web responsive de bază
- Exersarea tehnologiilor HTML & CSS

---

##  Structura proiectului

Proiectul conține următoarele fișiere:

```text
index.html        -> Pagina principală (About)
cv.html           -> Pagina CV (informații personale + skills)
Proiecte.html     -> Lista proiectelor
contact.html      -> Pagina de contact + formular
style.css         -> Stilizarea întregului website
